Eleve sua produtividade sem sair do editor! O **NoKanban** foi construído para desenvolvedores que precisam gerenciar pendências, manter o foco e sincronizar o status do projeto com a equipe, tudo isso sem quebrar o fluxo de trabalho e o _context switch_.

## ✨ Funcionalidades

- ✅ **Tarefas e Subtarefas:** Organize seu dia com níveis de prioridade (Alta, Média, Baixa) e quebre tarefas complexas em subtarefas.
- 🍅 **Timer Pomodoro Integrado:** Mantenha o foco absoluto com um cronômetro personalizável direto na barra lateral.
- ☁️ **Sincronização na Nuvem:** Salve seu progresso no **GitHub Gist** de forma segura e privada.
- 🤖 **Integração com Equipe:** Dispare o _Status Report_ do seu projeto direto para o **Discord** e **Telegram** com barras de progresso visuais gamificadas.
- 💾 **Auto-Backup:** Fique tranquilo com backups locais automáticos rodando em segundo plano.
- 📦 **Exportação Flexível:** Exporte suas anotações para JSON, Markdown ou formato de texto simples.

## 🚀 Como Instalar

1. Abra o VS Code.
2. Pressione `Ctrl+P` (Windows/Linux) ou `Cmd+P` (Mac) e digite:
   ```bash
   ext install nokanban-dev.nokanban
   ```
